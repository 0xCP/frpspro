# frps 一键安装脚本
## 支持系统 Centos 7+ Debian 8+
```
wget -N --no-check-certificate git.io/f.sh && chmod +x f.sh && bash f.sh install
```


## 一键修改 token
```
bash f.sh token
```


## 一键卸载 frps
```
bash f.sh uninstall
```


## 一键修改 bind_port
```
bash f.sh bind_port
```

